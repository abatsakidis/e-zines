=======================================================================
PART A ~ THE CONVERSION PROCESS
=======================================================================

~%~%~%~%~%~%~%~%~%~%~%~%~%~%~%
USING WINAMP TO CONVERT TO WAV (winamp.com)
~%~%~%~%~%~%~%~%~%~%~%~%~%~%~%

- Open your Winamp
- Right-click and go to Options - Preferences
- Under "Plug-ins" select Output
- Change the plug-in to "Nullsoft Disk Writer plugin-in v1.0"
- Then click Configure
- Change the directory to where you want to store the wavs
- Now play a Mp3 as you normally would, and it will convert them
  for you.

NOTE: TO RETURN WINAMP TO ITS NORMAL STATE CHANGE THE DISK WRITER 
      BACK TO "WAV_OUTPUT"

~%~%~%~%~%~%~%~
OTHER UTILITIES

=====================================================================
PART B ~ THE BURNING PROCESS
=====================================================================

Now since I have no clue what burner software you use, go contact your
booklet for information. Usually you get the option to create a music
CD at the beginning of your program launch.



~%~%~%~%~%~%~%~%~
NERO BURNER USERS
~%~%~%~%~%~%~%~%~

1.  Launch Nero Burner
2.  Select Audio-CD from the left menu
3.  Click New
4.  Use the File Explorer and find the newly created Wavs
5.  Move the Wavs over to the new CD Window
6.  Click File - Write Cd..
7.  Burning process should begin!
