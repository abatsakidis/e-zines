 Netscape Navigator 4.x
 ~%~%~%~%~%~%~%~%~%~%~%
 
 - Open Netscape
 - From the File Menu Click Edit - Point to Preferences
 - From the menu on the left select Navigator
 - Click clear History, Click OK to prompt
 - Click clear location bar, click ok to the prompt
 - Double-click on Advanced on the left menu - Select cache
 - Click Clear Memory Cache, Click OK to prompt
 - Click Clear Disk Cache, Click OK to prompt
 - Click OK

 Internet Explorer 5.x
 ~%~%~%~%~%~%~%~%~%~%~%
 
 - Open Internet Explorer
 - Click Tools from the File Menu - Select Internet Options
 - Click delete files under Temperary Internet Files
 - Select Delete Offline content - Click OK
 - Click Clear History, click OK to prompt
 - Click OK
